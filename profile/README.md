  ## Architecture

  ```
  Signal Detection → AI Recommendation → Human Approval → Execution → Cryptographic Proof
  ```

  - **AI governance by design** — advisory agents cannot execute consequential actions without human confirmation
  - **Proof-carrying execution** — every decision produces a tamper-evident, hash-chained receipt
  - **Multi-tenant isolation** — cross-tenant access is architecturally prevented, not policy-controlled
  - **Deny-by-default** — all API routes require authentication; public endpoints are explicitly allowlisted
