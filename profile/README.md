  ## What We Build

  SZL Holdings builds the **governed decision infrastructure layer** — the platform that connects what is observable to what is executable, under governance, with full attribution.

  **A11oy** is the governed agentic execution fabric. Eight domain intelligence packs — TENAX (cyber), Counsel (legal), PARAGON (defense), SEXTANT (maritime), DOMAINE (real estate), Carlota Jo (advisory), KORA (decision intelligence), and LUMINA (executive briefing) — extend the same governance infrastructure across verticals.

  Every AI recommendation carries source citations and confidence scores. Every consequential action requires human confirmation. Every outcome produces cryptographic proof.

  ## Trust & Governance

  | Concern | Approach |
  |:--------|:---------|
  | AI without oversight | Advisory agents require explicit human confirmation — enforced at the fabric layer |
  | Opaque AI outputs | All recommendations include source citations, confidence scores, and retrieval provenance |
  | Audit accountability | Every action generates an immutable audit event with actor attribution |
  | Access control | Role-based access control with org-scoped tenant isolation |
