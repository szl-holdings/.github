  ## About

  SZL Holdings builds governed, production-ready command software for organizations that cannot afford silent failures, invisible risk, or unaccountable AI. Every system ships with explicit audit trails, human-in-the-loop controls, and traceable decision paths.

  One TypeScript monorepo. One shared database schema. One execution engine. Six domain-specialized platforms — all compounding on the same architecture.

  ## Principles

  | Principle | What It Means |
  |:----------|:-------------|
  | **AI Governance by Design** | Advisory agents cannot execute consequential actions without explicit human confirmation. Enforced at the workflow layer — not just the UI. |
  | **Evidence-Backed Decisions** | Every AI recommendation includes source citations, confidence scores, and retrieval provenance. No opaque outputs. |
  | **Explicit Over Implicit** | Platform state is always visible. No silent fallbacks. Failures surface, not hide. |
  | **Shared Fabric** | Lyte and Alloy are platform infrastructure. Domain packs are structured applications built on the same foundation. Every improvement compounds across every platform. |
