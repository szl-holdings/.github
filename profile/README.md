# SZL Holdings

  > Governed decision infrastructure for enterprises.

  We build the operational layer for AI systems that need to be auditable by construction.

  ## What we ship

  - **[ouroboros](https://github.com/szl-holdings/ouroboros)** — bounded-loop runtime. v6.2.0, 172/172 tests.
  - **[ouroboros-thesis](https://github.com/szl-holdings/ouroboros-thesis)** — v3 paper, [DOI 10.5281/zenodo.19983066](https://doi.org/10.5281/zenodo.19983066).
  - **[a11oy](https://github.com/szl-holdings/a11oy)** — orchestration product with operator-facing decision intelligence and a working Claude-backed chat.
  - **[szl-holdings-platform](https://github.com/szl-holdings/szl-holdings-platform)** — pnpm monorepo: A11oy plus seven domain surfaces (Sentra, Amaru, Counsel, Terra, Vessels, Carlota Jo, plus the SZL API).

  The thesis is the citable proof. The runtime is the open-source primitive. A11oy is the unified governance surface. The seven domain surfaces are the verticals.

  ## Status

  Platform-development stage. Runtime is GA. Thesis is published with a DOI on Zenodo. Domain surfaces are working integrated artifacts in the monorepo, not yet deployed to production. SZL Holdings has engaged the Empire APEX Accelerator (administered by NYSTEC) for procurement counseling, which is advisory work, not a third-party audit.
